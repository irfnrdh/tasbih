# Tasbih

Ionic Vue 3 + Vite + Capacitor app.

## Replit Setup
- Workflow: `Start application` runs `npm run dev` on port 5000 (webview).
- Vite is configured with `host: 0.0.0.0`, `port: 5000`, and `allowedHosts: true` to work behind Replit's proxy.
- Deployment: static target, build `npm run build`, publish `dist/`.
